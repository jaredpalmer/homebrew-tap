# jaredpalmer/homebrew-tap

Homebrew tap for `mogcli`.

## Install

```bash
brew tap jaredpalmer/tap
brew install jaredpalmer/tap/mogcli
```

## Update Formula for a Release

1. Set `version` in `Formula/mogcli.rb`.
2. Update the four `sha256` values from the GitHub release `checksums.txt` file.
3. Commit and push to `main`.
